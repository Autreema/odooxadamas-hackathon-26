# Human Resource Management System

## Description
A Human Resource Management System (HRMS) built using React, TypeScript, and Vite. This application helps manage employees, attendance, leave requests, payroll, and dashboard statistics.

## Features
- Employee Management
- Attendance Tracking
- Leave Management
- Payroll Management
- Dashboard & Analytics
- Audit Logs

## Technologies Used
- React
- TypeScript
- Vite
- CSS

## Installation

1. Clone the repository:
```bash
git clone https://github.com/HARINI01-vaskl/Human-resource-management-system.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```
